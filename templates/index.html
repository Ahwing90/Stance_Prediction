Title: Predicting Political Stance Using Sentiment Analysis On Social Media

Project Description

This project aims to demonstrate the application of sentiment analysis techniques in predicting the political sentiment and stance, using the 2020 US Presidential Election as a case study. Data from online and social media, namely Twitter and Reddit, will be crawled on real-time and processed, in order to demonstrate an efficient sentiment prediction.


Text Preprocessing will be performed on the raw textual data, followed by sentiment analysis, in order to determine the level of political supports for the two main candidates from the bipartisan political systems in the USA. Besides, topic modelling is also performed on the data to determine the main topics happening in the US political arena.


Crawling, Data Preprocessing and Analysis are carried out in Python and Jupyter Notebook. Data storing is performed locally using MongoDB Database. Visualization and project deployment is done with local web pages (HTML) using FLASK Library in Python.
